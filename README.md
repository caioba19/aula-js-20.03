# 📘 Atividade Prática — Funções em JavaScript

Exercícios práticos de introdução a funções em JavaScript, utilizando `prompt`, `alert` e `document.writeln` para entrada e saída de dados.

---

## 📁 Estrutura da Atividade

| Nº | Função | Descrição |
|----|--------|-----------|
| 01 | `msg()` | Exibe uma mensagem de boas-vindas via `alert` |
| 02 | `peganome()` | Captura o nome do usuário e exibe uma saudação |
| 03 | `soma()` | Soma dois valores fixos e exibe o resultado |
| 04 | `soma2()` | Soma dois valores digitados pelo usuário |
| 05 | `soma3()` | Calcula a área de um triângulo (base × altura / 2) |
| 06 | `idade()` | Converte idade em anos para dias |
| 07 | `converterIdade()` | Converte idade em anos para meses |
| 08 | `media()` | Calcula a média de 3 notas e indica aprovação/reprovação |
| 09 | `calcularDesconto()` | Aplica desconto percentual sobre um preço |
| 10 | `tabuada()` | Exibe a tabuada de 1 a 10 de um número informado |
| 11 | `converterMedida()` | Converte metros para centímetros |

---

## 🚀 Como Executar

1. Crie um arquivo `index.html` e inclua o script JS:

```html
<!DOCTYPE html>
<html lang="pt-BR">
  <head>
    <meta charset="UTF-8" />
    <title>Funções JS</title>
  </head>
  <body>
    <script src="funcoes.js"></script>
  </body>
</html>
```

2. Abra o arquivo `index.html` no navegador.
3. As funções serão executadas em sequência, solicitando dados via `prompt`.

---

## 🧠 Conceitos Praticados

- Declaração e chamada de funções
- Entrada de dados com `prompt()`
- Saída de dados com `document.writeln()` e `alert()`
- Conversão de tipos com `parseInt()` e `parseFloat()`
- Operadores aritméticos (`+`, `-`, `*`, `/`)
- Estrutura condicional `if / else`
- Estrutura de repetição `for`

## 📌 Exemplos de Uso

### `calcularDesconto()`
```
Entrada:  Preço = 200 | Desconto = 15%
Saída:    Preço original: R$ 200.00
          Desconto (15%): R$ 30.00
          Preço final: R$ 170.00
```

### `tabuada()`
```
Entrada:  7
Saída:    7 x 1 = 7
          7 x 2 = 14
          ...
          7 x 10 = 70
```

### `converterMedida()`
```
Entrada:  3.5
Saída:    3.5 metro(s) = 350 centímetro(s)
```

### `media()`
```
Entrada:  Nota1 = 7 | Nota2 = 5 | Nota3 = 8
Saída:    6.666... O aluno foi aprovado
```
